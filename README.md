## 🧠 Exemplo de Código

### <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/python/python-original.svg" width="20"> Python 

```python
import pandas as pd

# Carregar dados
dados_carregados = pd.read_csv("dados.csv")

# Limpeza básica
dados_carregados = dados_carregados.dropna()

# Agrupamento
resultado = dados_carregados.groupby("categoria")["valor"].sum()

print(resultado)
```
